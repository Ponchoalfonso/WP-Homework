Medidas

CSS cuenta con diversas medidas que nos ayudan asignar valor de longitod a algunas propiedades de elementos, tales como width, height, margin, padding, font-size, etc.. Las longitudes generalmente se expresan con un número seguido de su tipo de medida, por ejemplo 16px.

Relativas:

Medida: Pixeles (px)
Un pixel equivale a 1/96 de pulgada, aunque puede variar a través de dispositivos, ya que en resoluciones pequeñas 1px equivale a un pixel físico del dispositivo y en resoluciones mas grandes 1px equivale a múltiples pixeles físicos.

Ejemplo: Div de 100px X 100px

Medida: (em)
Esta medida es relativa al tamaño de fuente, 2em equivalen a dos veces el tamaño de la fuente actual.

Ejemplo: Div de 5em X 5em

Medida: (ex)
Esta medida es relativa al alto de la X de la fuente actual.

Ejemplo: Div de 5ex X 5ex

Medida: (rem)
Esta medida es relativa al tamañod e fuene del elemento raiz del html.

Ejemplo: Div de 5rem X 5rem

Absolutas:

Medida: Pulgada (in)
Medida absoluta en pulgadas. Pulgada = 2.54cm = 96px.

Ejemplo: Div de 2in X 5in

Medida: Centímetro (cm)
Medida absoluta en centímetros.

Ejemplo: Div de 2cm X 5cm

Medida: Milímetro (mm)
Medida absoluta en milímetros.

Ejemplo: Div de 60mm X 60mm

Medida: Punto (pt)
Un putno equivale a 1/72 de pulgada.

Ejemplo: Div de 5pt X 5pt

Medida: Pica (pc)
Una pc equivale a 12pt.

Ejemplo: Div de 5pc X 5pc

Porcentuales:

Medida: Porciento (%)
Relativo en porcentaje al elemento padre.

Ejemplo: Div de 50% X 50%

Colores

En CSS podemos asiganr colores a elementos a trabes de ciertas propiedades. Para dar valores de color tenemos las siguientes etiquetas.

Etiquetas para asignar valores de colores

Etiqueta: rgb
Esta etiqueta nos permite ingresar valores a los canales Red Green y Blue para crear colores.
La entrada de datos puede ser porcentual, decimal o hexadecimal.

Ejemplo rgb(194, 124, 124)

Ejemplo #5c7aa7

Ejemplo rgb(20%, 50%, 18%)
